# Driveway Gates Surrey — silo plan

Live site, reconciliation and gap-fill. UK English, £ GBP; the lead form is the only
CTA. Titles are provisional — SERP/PAA-validate before writing. Canonical host uses
`www.`; location pages at `/location/`.

## Local focus (what this site is built to own)

1. **Prestige gated-estate access-control integration** — ANPR / video intercom /
   CCTV matched to the culture of Wentworth, St George's Hill, Oxshott, Crown Estate
   and the Elmbridge golden triangle. This site builds its strongest cluster around
   the "gate access control / ANPR for home" intent.
2. **Top-of-market budgets** — estate-grade £15k+ installs; cost content anchors
   higher than the national £3k-£7k.
3. **Surrey Hills AONB + 200+ conservation areas + 6,000+ listed buildings** —
   design constraints steering timber/estate-character gates.
4. **Sloped North Downs sweeping gravel drives** — gradient, hinge geometry, motor
   spec, long cable runs.
5. **Equestrian / off-mains rural plots** (Cranleigh/Dunsfold/Shere) — solar vs SWA
   trenching, 75m+ runs.
6. **Affluent-area perimeter-security framing** — Elmbridge burglary clusters;
   security-led not kerb-appeal-led.

## Priority spokes — lead clusters first

**Estate access-control (flagship):** ANPR gates for private estates; video intercom
for large detached homes (staff/deliveries/contractors); integrating gates with
existing estate CCTV/alarm; smart-home gate control.

**AONB / heritage:** wooden gates in a conservation area / AONB; underground vs ram
motors (concealed for heritage frontages); oak vs iroko vs Accoya; bespoke
wrought-iron estate gates (crest/monogram).

**Sloped / large drives:** sliding gates for sloped driveways; cantilever vs tracked
for long gravel runs; wind load on wide estate gates.

**Off-mains:** solar vs mains power for equestrian/rural plots 75m+ from supply.

## One query per page (no internal cannibalisation)

Every guide and spoke owns exactly one reserved primary query; no two pages target
the same query. This site's strongest national-query bids are access-control/ANPR,
best gate motor brands (estate-install context), and "do electric gates add value"
(top property values give the hook). Cede the bare national "electric gates cost UK"
to aggregators; target only Surrey estate-tier cost intent.
